# 🧑‍💻 TeamType — Real-time Collaborative Code Editor

TeamType is a full-stack collaborative coding platform where developers can create or join sessions and write code together in real-time. It features a beautiful frontend UI, a scalable backend with WebSockets, and live code sharing capabilities.

---

## 📁 Project Structure

```
teamtype/
├── client/     # Next.js frontend
└── server/     # Express backend
```

---

## 🔗 Repositories

- 🎨 **Frontend** → [`/client`](./client)
- 🧠 **Backend** → [`/server`](./server)

> Each folder contains its own README with setup instructions and details.

## 🚀 Quick Start

### 1. Clone the Monorepo

```bash
git clone https://github.com/your-username/teamtype.git

```

Refer to their individual READMEs:

- [`client/README.md`](./client/README.md)
- [`server/README.md`](./server/README.md)

## ✨ Features

- Real-time collaboration with Socket.IO
- Syntax-highlighted code editor (CodeMirror)
- Responsive frontend with Tailwind & shadcn
- REST API + WebSocket API
- Scalable architecture with clean separation
